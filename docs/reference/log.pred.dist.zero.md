# A function to calculate log predictive distribution (pmf and cdf) of logistic component of a 'brm' fit

A function to calculate log predictive distribution (pmf and cdf) of
logistic component of a 'brm' fit

## Usage

``` r
# S3 method for class 'pred.dist.zero'
log(fit)
```

## Arguments

- fit:

  A \`brm\` fit.
