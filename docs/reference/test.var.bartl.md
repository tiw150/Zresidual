# Bartlett test core (internal): variance homogeneity across bins/levels

Bartlett test core (internal): variance homogeneity across bins/levels

## Usage

``` r
test.var.bartl(Zresidual, fitted.value, k.bl = 10)
```

## Arguments

- Zresidual:

  numeric vector

- fitted.value:

  numeric/factor vector (same length as Zresidual)

- k.bl:

  number of bins for numeric covariates

## Value

p-value (numeric) or NA_real\_
