# A function to calculate ANOVA of Zresidual

A function to calculate ANOVA of Zresidual

## Usage

``` r
aov.test.zresid(Zresidual, X = c("lp", "covariate"), k.anova = 10)
```

## Arguments

- Zresidual:

  A z-residual.

- X:

  Linear predictor or covariate

- k.anova:

  Number of bins if applicable
