# A function to draw scatter plot of a Deviance Residual

A function to draw scatter plot of a Deviance Residual

## Usage

``` r
# S3 method for class 'dev.resid'
plot(
  Deviance.residual,
  ylab = "Deviance Residual",
  X = c("index", "lp", "covariate"),
  main.title = "Deviance Residual Plot",
  outlier.return = FALSE,
  ...
)
```
