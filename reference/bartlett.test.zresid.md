# A function to calculate Bartlett of Zresidual

A function to calculate Bartlett of Zresidual

## Usage

``` r
# S3 method for class 'zresid'
bartlett.test(Zresidual, X = c("lp", "covariate"), k.bl = 10)
```

## Arguments

- Zresidual:

  A Z-residual.

- X:

  Linear predictor or covariate

- k.bl:

  Number of bins if applicable
