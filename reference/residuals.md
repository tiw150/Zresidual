# A function to calculate other type of residual

A function to calculate other type of residual

## Usage

``` r
residuals(
  fit.object,
  data,
  residual.type = c("censored Z-residual", "Cox-Snell", "martingale", "deviance")
)
```

## Value

- censored.Zresidcensored Z-residual

- ucsunmodified CS residuals

- martgMartingale residuals

- devDeviance residuals
