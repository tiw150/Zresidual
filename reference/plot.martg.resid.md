# A function to draw scatter plot of a Martingale Residual

A function to draw scatter plot of a Martingale Residual

## Usage

``` r
# S3 method for class 'martg.resid'
plot(
  Martingale.residual,
  ylab = "Martingale Residual",
  X = c("index", "lp", "covariate"),
  main.title = "Martingale Residual Plot",
  outlier.return = FALSE,
  ...
)
```
