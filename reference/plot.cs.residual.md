# A function to draw scatter plot of a Cox-snell Residual

A function to draw scatter plot of a Cox-snell Residual

## Usage

``` r
# S3 method for class 'cs.residual'
plot(
  cs.residual,
  ylab = "Cumulative Hazard Function",
  main.title = "Cox-Snell Residuals Scatterplot",
  outlier.return = FALSE,
  ...
)
```

## Arguments

- cs.residual:

  A numeric vector of Cox-Snell residuals.
